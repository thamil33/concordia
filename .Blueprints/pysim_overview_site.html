Framework Overview

Pysim is a modular Python framework for creating generative agent-based models. It enables complex simulations where AI-driven entities interact within a defined environment. This dashboard provides an interactive exploration of its core concepts and operational flow.

Core Systems

🗣️ Language Model Integration

Pysim abstracts LLM interactions through a common `LanguageModel` interface, allowing for pluggable backends like local servers (LM Studio, Ollama) or APIs (OpenRouter).

🧠 Associative Memory

Entities remember information using an `AssociativeMemoryBank`. It converts text to vector embeddings for semantic retrieval, enabling memory recall based on conceptual similarity, not just keywords.

⏰ Clock & 🔄 Engine

The `GameClock` manages time progression. The `SequentialEngine` drives the simulation, orchestrating the turn-based loop of observation, action, and event resolution.

🤝 Component Relationships

Simulation Assembly

Pysim simplifies simulation creation using a `Configurator` and `Prefabs`. This system abstracts away the complex wiring of components, allowing you to define a simulation through high-level configuration.
